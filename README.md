## Overview

This project focuses on predicting flight delays using an **Attention-Based Deep Neural Network (DNN)** trained on real-world aviation data.

The model captures complex relationships between multiple factors such as:
- Weather conditions  
- Flight schedules  
- Operational constraints  

and dynamically identifies the most important features influencing delays.

---

## Problem Statement

Flight delays are influenced by multiple interdependent variables, making them difficult to predict using traditional models.

This project addresses the problem using:
- Deep learning  
- Feature-level attention mechanisms  
- Real-world aviation datasets  

to improve prediction accuracy and interpretability.

---

## Approach

- Data preprocessing & feature engineering  
- Handling missing values  
- Training an Attention-based DNN  
- Binary classification:
  - Delayed  
  - On-time  

---

## Model Highlights

- Attention mechanism to weigh important features  
- Dense neural network architecture  
- Improved interpretability over standard models  
- Robust performance on real-world data  

---

## Results

- Achieved strong prediction performance  
- Balanced precision and recall  
- Effective classification of delayed vs on-time flights
